# Features

  - Login and Register
  - Deposit / Withdraw
  - Query API

# Libraries Used

  - bootstrap 4
  - express
  - express-redirect
  - cookie-parser
  - ejs
  - mongoose

# Installation
Make a project folder then open it:
```sh
$ mkdir nodebank && cd nodebank
```

Clone Repository:
```sh
$ git clone git@github.com:jeffreynerona/node-bank.git
```

Install dependencies:
```sh
$ npm install
```

Run the server:
```sh
$ npm start
```
